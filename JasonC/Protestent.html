Les Manifestants

Récemment, il y a eu beaucoup de fusillades. Malheureusement, beaucoup d'entre eux sont dans les écoles. Pour cette raison, beaucoup d’étudiants protestent contre les lois sur les armes à feu. À Des Moines, la capitale de notre état, beaucoup d’étudiants protestent pour changer les lois sur les armes à feu et imposer des restrictions sur les armes utilisées dans ces fusillades. Les étudiants ont bravé le froid pour essayer de changer. La participation politique est un énorme rôle pour être étudiant ici! Même si vous ne vous impliquez pas, cela aide à rester informé. Les étudiants essaient vraiment de faire un changement dans le monde.
